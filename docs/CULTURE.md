# Saiteki文化の客観的指標 (Saiteki Culture Indicators)

`TEAM.md` に蓄積された全社員の人柄分析データをもとに、株式会社Saitekiの企業文化を客観的な指標として抽出・定義しました。

## 1. パーソナリティ構成比 (Big 5 分析)

全社員の分析結果を集計した結果、Saitekiには際立った「性格的傾向」が存在します。これがSaitekiのカルチャーの基盤となっています。

| 指標 | 平均スコア (目安) | Saitekiカルチャーの特徴 |
| :--- | :---: | :--- |
| **協調性 (Agreeableness)** | **9.0 / 10** | **極めて高い**。他者へのリスペクト、感謝、支援が自然に行われる「心理的安全性」の高い組織。<br>対立よりも融和と協力を重んじる。 |
| **開放性 (Openness)** | **8.9 / 10** | **極めて高い**。新しい技術(AI等)、未知の体験、多様な価値観に対する好奇心が強い。<br>変化を恐れず、常に学習し続ける「知的好探求心」の集団。 |
| **誠実性 (Conscientiousness)** | **7.6 / 10** | **高い**。責任感が強く、タスクを完遂する力がある。真面目で信頼できるメンバーが多い。 |
| **外向性 (Extraversion)** | **7.5 / 10** | **やや高い**。社交的だが、単なるお喋り好きではなく「チームへの関与」としてのコミュニケーションを重視する傾向。<br>少数での深い対話を好むメンバーも共存。 |
| **神経症的傾向 (Neuroticism)** | **3.5 / 10** | **低い**。感情が安定しており、ポジティブ。困難な状況でも冷静かつ前向きに捉えるレジリエンスがある。 |

### 💡 文化的インサイト: 「学習する調和型集団」
Saitekiは**「高い知的好奇心(Openness)」**と**「高い協調性(Agreeableness)」**が共存しているのが最大の特徴です。
単に仲が良いだけでなく、**「共に新しいことを学ぶ」「知識を共有し合う」**ことによって繋がりが強化される文化があります。

## 2. 共有されているコアバリュー (共通の価値観)

メンバーの「価値観とモチベーター」分析から抽出された、頻出するキーワードと行動原理です。

### ① 貢献と感謝 (Contribution & Gratitude)
*   **キーワード**: 「チームへの貢献」「他者からの感謝」「役に立ちたい」
*   **行動規範**: 自分のスキルが誰かの役に立つことに最大の喜びを感じる。互いに「ありがとう」を伝え合う頻度が高い。

### ② 自律的学習と共有 (Learning & Sharing)
*   **キーワード**: 「新しい技術」「AI活用」「ナレッジ共有」「成長」
*   **行動規範**: 誰かに言われて学ぶのではなく、自らの好奇心で学び、得た知識を惜しみなくチームにシェアする（Giveの精神）。

### ③ 変化への適応とAI活用 (Adaptability & AI First)
*   **キーワード**: 「AI駆動開発」「業務効率化」「新しい環境」
*   **行動規範**: 変化をポジティブに捉える。AIなどの新技術を「ツール」として使いこなし、自らの業務を進化させることに躊躇がない。

## 3. Saiteki "S-Class" 人材の定義

データから逆算される、Saitekiにおいて「S級」と評価される人物像（モデル）は以下の通りです。

1.  **知的好奇心の塊である** (Openness 9+)
    *   未知の領域やAI技術に対して、恐怖ではなく「ワクワク」を感じて飛び込める。
2.  **圧倒的なGiverである** (Agreeableness 9+)
    *   自分の成果だけでなく、チーム全体の成功を本気で喜べる。知識を独占せず共有する。
3.  **ポジティブな安定感** (Neuroticism < 4)
    *   変化の激しい環境でも、精神的に安定し、周囲を不安にさせない。
